# Linux
AIStor supports deploying resources onto Linux infrastructures and provides packages for Red Hat Enterprise Linux and Ubuntu Server platforms.
Setting up AIStor on Linux involves three steps:
  1. [Installing AIStor](https://docs.min.io/enterprise/aistor-object-store/installation/linux/install/)
  2. [Setting up network encryption](https://docs.min.io/enterprise/aistor-object-store/installation/linux/network-encryption/)
  3. [Setting up server-side encryption](https://docs.min.io/enterprise/aistor-object-store/installation/linux/server-side-encryption/)
