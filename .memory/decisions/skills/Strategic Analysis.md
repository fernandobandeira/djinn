# Strategic Analysis

**Tier:** Domain

## Core Principle

**See the whole board before moving.** Strategic decisions require understanding all forces at play—your position, competitors, and possible futures.

## Problem

Strategic decisions lack structured frameworks:
- Important factors get missed
- Competitive dynamics aren't understood
- Future scenarios aren't explored
- Decisions made without full context

## Solution

Strategic frameworks that ensure comprehensive analysis.

**Techniques:**
- **SWOT** - Strengths, Weaknesses, Opportunities, Threats
- **Porter's Five Forces** - Competitive dynamics analysis
- **Scenario Planning** - Explore multiple futures
- **What-If Analysis** - Test sensitivity to assumptions
