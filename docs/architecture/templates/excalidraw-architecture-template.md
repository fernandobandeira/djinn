# Architecture Diagram Template

==⚠  Switch to EXCALIDRAW VIEW in the MORE OPTIONS menu of this document. ⚠==

## Component Library for Architecture Diagrams

This template includes common shapes and patterns for architecture diagrams:

### System Components
- **Services**: Rounded rectangles with tech stack labels
- **Databases**: Cylinder shapes with DB type
- **Message Queues**: Hexagon shapes
- **External APIs**: Cloud shapes
- **Load Balancers**: Diamond shapes

### Connections
- **Synchronous**: Solid arrows
- **Asynchronous**: Dashed arrows  
- **Data Flow**: Thick arrows with labels
- **Dependencies**: Dotted lines

### Color Scheme
- **Frontend**: Blue (#4A90E2)
- **Backend**: Green (#5CB85C)
- **Database**: Orange (#F0AD4E)
- **External**: Purple (#9B59B6)
- **Infrastructure**: Gray (#6C757D)

### Text Styles
- **Component Names**: Bold, 16pt
- **Tech Stack**: Italic, 12pt
- **Annotations**: Regular, 10pt

## Usage
1. Duplicate this template
2. Use the component library 
3. Maintain consistent colors
4. Add tech stack labels
5. Include data flow directions
