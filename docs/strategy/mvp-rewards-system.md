# Simplified Earning Structure

## Base Points (Every Receipt)
- **Standard Scan**: 100 points
- **First Scan of Day**: 200 points (2x bonus)
- **Weekend Scan**: 150 points (1.5x bonus)

## Streak Bonuses (Habit Building)
- **3-day streak**: +200 points
- **7-day streak**: +500 points
- **30-day streak**: +2,000 points
- **66-day streak**: +5,000 points (habit formed!)

## Quick Achievements (Instant Gratification)
- **First Receipt**: 500 points (welcome bonus)
- **Category Complete**: 300 points (scan from each category)
- **Speed Scanner**: 200 points (5 receipts in 2 minutes)
- **Week Complete**: 1,000 points (at least 1 scan per day)

# Monthly Value Calculation

## Casual User (5 scans/month)
- Base points: 500
- Bonuses: ~250
- **Total: 750 points = $0.75 value**
- Can redeem: 1 Basic AI Wish

## Regular User (20 scans/month)
- Base points: 2,000
- Streak bonus: 500
