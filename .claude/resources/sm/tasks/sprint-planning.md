# Sprint Planning Task Guidelines

## Objectives
- Define sprint goal
- Select and prioritize user stories
- Ensure team capacity alignment
- Create sprint backlog

## Facilitation Checklist
1. Review product backlog
2. Team capacity assessment
3. Story point estimation
4. Commitment to sprint scope
5. Identify potential risks

## Collaboration Techniques
- Use planning poker for estimation
- Encourage team dialog
- Break down complex stories
