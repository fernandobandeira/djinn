# Retrospective Facilitation Guidelines

## Meeting Structure
1. Set the stage
2. Gather data
3. Generate insights
4. Decide what to improve
5. Close the retrospective

## Key Facilitation Techniques
- Create safe environment
- Encourage honest feedback
- Focus on continuous improvement
- Avoid blame culture
- Track action items

## Reflection Prompts
- What went well?
- What could be improved?
- What will we commit to changing?
