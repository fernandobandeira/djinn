# PM Elicitation Methods

## Core Product Methods

### Expand or Contract for Stakeholders
- Adjust detail level based on audience (executives vs engineers)
- Simplify for executive summaries
- Elaborate for technical implementation teams
- Tailor language to stakeholder expertise

### Market Validation
- Compare against competitor features
- Validate with market research data
- Check against industry benchmarks
- Assess market fit and timing

### User Journey Mapping
- Walk through complete user flows
- Identify friction points
- Discover missed requirements
- Validate feature completeness

## Strategic Analysis Methods

### Business Impact Assessment
- Quantify revenue potential
- Calculate cost savings
- Measure efficiency gains
- Evaluate strategic alignment

### Risk-Opportunity Analysis
- Identify business risks
- Spot market opportunities
- Assess competitive threats
- Evaluate timing considerations

### MVP Scope Challenge
- Apply "What's truly minimum?"
- Challenge each feature inclusion
- Identify true differentiators
- Focus on core value proposition

## Requirements Refinement

### Acceptance Criteria Deep Dive
- Make criteria measurable
- Add edge cases
- Include performance requirements
- Define clear success metrics

### Technical Feasibility Check
- Validate with engineering
- Identify technical constraints
- Assess implementation complexity
- Consider technical debt

### Cross-Functional Impact
- Marketing implications
- Sales enablement needs
- Support requirements
- Operations impact

## Prioritization Methods

### Value vs Effort Matrix
- Plot features on 2x2 grid
- Identify quick wins
- Spot value traps
- Guide prioritization decisions

### MoSCoW Analysis
- Must have features
- Should have features
- Could have features
- Won't have (this time)

### RICE Scoring
- Reach: How many users affected
- Impact: How much value per user
- Confidence: How certain are we
- Effort: How much work required

## Stakeholder Alignment

### Stakeholder Round Table
- Gather perspectives from all stakeholders
- Identify conflicting priorities
- Find consensus points
- Document disagreements

### Executive Pitch Test
- Create 30-second elevator pitch
- Test value proposition clarity
- Validate business case
- Ensure strategic alignment

### Customer Advisory Board
- Get direct customer input
- Validate assumptions
- Prioritize features
- Understand usage patterns

## Innovation Methods

### Competitive Differentiation
- What competitors can't/won't do
- Identify unique value
- Find market gaps
- Create moats

### Future-Back Planning
- Start from 3-year vision
- Work backwards to MVP
- Identify critical path
- Plan evolution stages

### Adjacent Possibilities
- Explore feature combinations
- Identify platform opportunities
- Consider ecosystem plays
- Plan for extensibility

## Validation Techniques

### Prototype Testing
- Create low-fi mockups
- Test with users
- Validate assumptions
- Iterate quickly

### A/B Test Planning
- Define test hypotheses
- Plan measurement approach
- Set success criteria
- Design experiments

### Metrics Definition
- Define leading indicators
- Set lagging indicators
- Create measurement plan
- Establish baselines

## Process Optimization

### Sprint Planning Prep
- Break down epics
- Size stories
- Identify dependencies
- Plan sprint goals

### Retrospective Mining
- Extract learnings from past projects
- Identify patterns
- Apply lessons learned
- Avoid repeated mistakes

### Continuous Discovery
- Regular customer interviews
- Ongoing market research
- Competitive monitoring
- Trend analysis

## Communication Methods

### Storytelling Framework
- Set context (problem)
- Introduce hero (user)
- Present challenge
- Show transformation (solution)
- Demonstrate value

### Data-Driven Narrative
- Lead with metrics
- Support with anecdotes
- Show trends
- Project impact

### Visual Communication
- Create journey maps
- Design process flows
- Build feature matrices
- Develop roadmap visuals

## Decision Support

### Decision Tree Analysis
- Map decision points
- Evaluate outcomes
- Assess probabilities
- Guide choices

### Opportunity Cost Assessment
- What are we NOT doing
- Resource trade-offs
- Timeline implications
- Strategic alternatives

### Pivot or Persevere
- Evaluate current trajectory
- Assess pivot options
- Consider sunk costs
- Make go/no-go decisions
