# Insight Synthesis Flow Protocol

## Core Principles
- Detect emergent patterns in ongoing analysis
- Synthesize patterns into actionable insights
- Integrate feedback to refine understanding
- Maintain coherence across multiple research streams

## Activation Triggers

1. **Pattern Emergence**
   - Recurring themes across multiple data sources
   - Cross-domain correlations identified
   - Threshold of supporting evidence reached

2. **Analysis Maturity**
   - Sufficient research depth achieved
   - Multiple perspectives gathered
   - User requests synthesis

3. **Contradiction Resolution**
   - Conflicting findings identified
   - Need for integrated perspective
   - Opportunity for reframing

## Process Flow

### 1. Pattern Detection
- Monitor ongoing analysis for emergent themes
- Track cross-cutting patterns
- Identify supporting and contradicting evidence
- Assess pattern confidence and scope

### 2. Delegation to Insight-Synthesizer
- Prepare pattern documentation
- Provide supporting evidence
- Define synthesis scope
- Specify output expectations

### 3. Pattern Presentation
- Communicate synthesized patterns clearly
- Provide confidence levels
- Highlight key supporting evidence
- Suggest implications and next steps

### 4. Feedback Integration
- Gather user validation and critique
- Identify pattern refinements needed
- Flag areas for deeper exploration
- Update working hypothesis

### 5. Iteration
- Refine patterns based on feedback
- Conduct targeted follow-up research
- Re-synthesize as new evidence emerges
- Track evolution of insights

## Success Criteria

- Pattern clarity and coherence
- Evidence sufficiency and quality
- User understanding and buy-in
- Actionable implications identified
- Integration with main narrative

## Integration Points

### With Interactive Facilitation
- Present patterns as options for deeper exploration
- Allow user-guided pattern refinement
- Support iterative discovery

### With Research Delegation
- Trigger targeted research to validate patterns
- Request deeper investigation of anomalies
- Support hypothesis testing

### With Main Analysis
- Update working hypotheses
- Adjust confidence levels in key findings
- Refine strategic implications
- Support recommendation development

## Output Format

Synthesized insights should include:
- Pattern statement (concise, clear)
- Scope (domains, timeframes affected)
- Supporting evidence (data, sources, logic)
- Confidence level (high, medium, low)
- Implications (strategic, operational, tactical)
- Recommended actions or next steps
- Uncertainties and limitations
