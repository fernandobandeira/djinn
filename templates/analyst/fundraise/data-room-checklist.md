# Data Room Checklist

Essential documents for investor due diligence, organized by category and stage.

## Core Principle

**Your pitch deck gets you in the door - your data room closes the deal.** 89% of investors now require secure digital access to due diligence materials. A well-prepared data room speeds up the process and builds trust.

---

## Organization Best Practices

### Folder Structure
```
/01-Company-Overview
/02-Financials
/03-Legal-Corporate
/04-Team
/05-Product
/06-Customers
/07-Market
/08-Operations
```

### Naming Conventions
- Use dates: `2025-Q1-Financial-Statements.pdf`
- Version control: `Cap-Table-v3-2025-01.xlsx`
- Consistent format across all files

### Access Control
- Grant read-only by default
- Track who views what
- Progressive disclosure (more docs after term sheet)

## Stage 1: Pre-Term Sheet (Initial Diligence)

Lighter documentation for initial investor evaluation.

### Company Overview
| Document | Description | Priority |
|----------|-------------|----------|
| Pitch Deck | Latest version, aligned with verbal pitch | Required |
| One-Pager | Concise business summary | Recommended |
| Company Overview | Mission, history, key milestones | Required |

### Financials (Summary)
| Document | Description | Priority |
|----------|-------------|----------|
| Financial Summary | High-level P&L, key metrics | Required |
| Cap Table | Current ownership structure | Required |
| Use of Funds | Planned allocation of raise | Required |

### Team
| Document | Description | Priority |
|----------|-------------|----------|
| Founder Bios | Backgrounds and relevant experience | Required |
| Org Chart | Current team structure | Recommended |

### Traction
| Document | Description | Priority |
|----------|-------------|----------|
| Key Metrics Dashboard | MRR, growth, retention, etc. | Required |
| Customer List (Summary) | Top customers, anonymized if needed | Recommended |

## Stage 2: Post-Term Sheet (Full Diligence)

Comprehensive documentation for closing.

### Financial Documents

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| P&L Statements | Last 3 years or since inception | Required | Required |
| Balance Sheet | Current snapshot | Required | Required |
| Cash Flow Statement | Historical and projected | Recommended | Required |
| Financial Projections | 3-5 year forecast | Required | Required |
| Monthly Actuals | Detailed monthly financials | Recommended | Required |
| Financial Model | Full model with assumptions | Required | Required |
| Bank Statements | 12 months | Upon request | Required |
| Debt/Loan Agreements | Any existing obligations | If applicable | Required |

### Cap Table & Equity

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Cap Table | Fully diluted ownership | Required | Required |
| Option Pool | Current and proposed | Required | Required |
| Stock Option Agreements | Template and granted | Recommended | Required |
| Previous Financing Docs | SAFEs, convertible notes, priced rounds | Required | Required |
| Investor Rights | Any existing rights | If applicable | Required |

### Legal & Corporate

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Certificate of Incorporation | Current articles | Required | Required |
| Bylaws | Corporate bylaws | Required | Required |
| Board Minutes | Historical meeting minutes | Recommended | Required |
| State Registrations | Good standing certificates | Recommended | Required |
| IP Assignments | All employee/contractor | Required | Required |
| Patents/Trademarks | Filed or granted | If applicable | Required |
| Material Contracts | Key agreements > $50K | Recommended | Required |
| Insurance Policies | D&O, E&O, general liability | Recommended | Required |

### Employment & Team

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Org Chart | Full team structure | Required | Required |
| Executive Bios | Detailed backgrounds | Required | Required |
| Employment Agreements | Key employees | Recommended | Required |
| Contractor Agreements | Key contractors | Recommended | Required |
| Employee Handbook | Policies and procedures | Optional | Recommended |
| Compensation Summary | Salary bands, equity | Upon request | Required |

### Product & Technology

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Product Overview | Features, screenshots, demo | Required | Required |
| Product Roadmap | 12-18 month plan | Required | Required |
| Technical Architecture | System design overview | Recommended | Required |
| Security Documentation | SOC2, penetration tests | If applicable | Recommended |
| IP Summary | Patents, trade secrets | If applicable | Required |

### Customers & Market

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Customer List | With contract details | Anonymized OK | Required |
| Customer Contracts | Template and key agreements | Sample | Required |
| Case Studies | Success stories | Recommended | Required |
| Customer References | Contacts for calls | 2-3 | 5+ |
| Pipeline Report | Current opportunities | Required | Required |
| Churn Analysis | Historical and reasons | If applicable | Required |
| NPS/Satisfaction Data | Customer feedback | Recommended | Required |

### Market & Competition

| Document | Description | Seed | Series A |
|----------|-------------|------|----------|
| Market Analysis | TAM/SAM/SOM with methodology | Required | Required |
| Competitive Analysis | Matrix and positioning | Required | Required |
| Go-to-Market Plan | Sales and marketing strategy | Required | Required |

## Seed Round Checklist

### Essential (Must Have)
- [ ] Pitch deck (latest version)
- [ ] One-pager summary
- [ ] Cap table (fully diluted)
- [ ] Financial model with projections
- [ ] Key metrics dashboard
- [ ] Founder bios
- [ ] Certificate of incorporation
- [ ] IP assignment agreements
- [ ] Previous financing docs (SAFEs, etc.)

### Recommended (Should Have)
- [ ] P&L since inception
- [ ] Customer list (can be anonymized)
- [ ] Product demo or screenshots
- [ ] Org chart
- [ ] Competitive analysis
- [ ] Go-to-market plan

### Upon Request
- [ ] Bank statements
- [ ] Employment agreements
- [ ] Technical documentation

---

## Series A Checklist

### Essential (Must Have)
- [ ] Pitch deck (latest version)
- [ ] Executive summary
- [ ] Cap table with option pool analysis
- [ ] Full financial model
- [ ] P&L, balance sheet, cash flow (3 years or since inception)
- [ ] Monthly financial actuals (12+ months)
- [ ] Key metrics with cohort analysis
- [ ] Customer list with contract values
- [ ] 5+ customer references
- [ ] Founder and executive bios
- [ ] Org chart
- [ ] All corporate documents (cert of inc, bylaws, minutes)
- [ ] IP assignments for all team members
- [ ] All previous financing documents
- [ ] Material contracts

### Recommended (Should Have)
- [ ] Product roadmap
- [ ] Technical architecture overview
- [ ] Security documentation
- [ ] Employee handbook
- [ ] Compensation summary
- [ ] Churn analysis
- [ ] Case studies (3-5)
- [ ] Market research and methodology

### Upon Request
- [ ] Bank statements (12 months)
- [ ] All employment agreements
- [ ] All contractor agreements
- [ ] Insurance policies
- [ ] State registrations
- [ ] Detailed customer contracts

## Document Quality Standards

### Financials
- Prepared by accountant or CFO
- Clear assumptions documented
- Consistent formatting
- Reconciles to bank statements

### Legal
- Reviewed by counsel
- Signed and executed
- No missing signatures or dates
- Current and in good standing

### Metrics
- Definitions provided
- Consistent calculation methodology
- Auditable source data
- Historical trends included

---

## Common Red Flags

| Issue | Why It Matters |
|-------|----------------|
| Missing IP assignments | Ownership disputes later |
| Messy cap table | Signals operational chaos |
| No board minutes | Governance concerns |
| Inconsistent financials | Trust issues |
| Missing contracts | Due diligence delays |
| Outdated documents | Shows lack of preparation |

## Timeline Expectations

| Stage | Typical Duration |
|-------|------------------|
| Initial review | 1-2 weeks |
| Deep diligence | 2-4 weeks |
| Legal review | 2-4 weeks |
| Total (Seed) | 4-8 weeks |
| Total (Series A) | 6-12 weeks |

**Pro tip**: Start preparing your data room 3-6 months before you plan to raise.

## Tools & Platforms

### Virtual Data Room Options
- DocSend (popular for early stage)
- Carta (integrated with cap table)
- Google Drive (budget option)
- Dropbox (simple sharing)
- Dedicated VDRs (Intralinks, etc.)

### Key Features to Look For
- View tracking and analytics
- Permission controls
- Watermarking
- Easy updates
- Clean sharing links
